# MadMachine

## Install

执行 `cnpm i` 安装所有依赖

## Available Scripts

In the project directory, you can run:

### 1. 启动项目

`npm run start`

### 2. 打包所有项目

`npm run app:publish`

### 3. 编译配置文件

`npm run config:build`
